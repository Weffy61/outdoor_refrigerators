    Инструкция по загрузке фотоотчета

    Для iPhone

    1. Сделайте фото холодильного оборудования, а также фото с его серийным номером.
    2. Откройте приложение «Фото» на вашем iPhone.
    3. Выберите фотографии, которые вы хотите загрузить.
    4. Нажмите на иконку «Поделиться» в левом нижнем углу экрана.
    5. Выберите «Сохранить в Файлы» и выберите папку для сохранения.
    6. Перейдите на сайт и откройте форму загрузки фотоотчета.
    7. Нажмите кнопку «Выберите файл» и выберите сохраненную фотографию из приложения «Файлы». Повторите действия для второй фотографии. Отправку осуществлять только данным способом, в противном случае Ваш отчет может быть не принят.
    8. Убедитесь, что фотографии успешно загрузились, и нажмите кнопку "Отправить", чтобы отправить отчет.
    9. После успешной отправки отчета вы будете перенаправлены в "Мои отчеты", где у отчета будет статус "На рассмотрении".

    Для Android

    1. Сделайте фото холодильного оборудования, а также фото с его серийным номером. Убедитесь, что у вас включено сохранение данных о местоположении в настройках фото, в противном случае ваш отчет может быть отклонен.
    2. Перейдите на сайт и откройте форму загрузки фотоотчета.
    3. Нажмите кнопку «Выберите файл» и выберите фотографию из файлов. Повторите действия для второй фотографии.
    4. Убедитесь, что фотографии успешно загрузились, и нажмите кнопку "Отправить", чтобы отправить отчет.
    5. После успешной отправки отчета вы будете перенаправлены в "Мои отчеты", где у отчета будет статус "На рассмотрении".
